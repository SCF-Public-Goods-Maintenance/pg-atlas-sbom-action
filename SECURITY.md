# Security Policy

## Reporting a Vulnerability

Please do not report security vulnerabilities via GitHub Issues.

Report vulnerabilities by opening a
[security advisory](https://github.com/SCF-Public-Goods-Maintenance/pg-atlas-sbom-action/security/advisories/new)
on this repository (preferred), and/or by messaging `@alioli` on Keybase.

We aim to respond to security reports within 5 business days.
